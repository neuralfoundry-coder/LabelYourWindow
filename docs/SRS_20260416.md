# Software Requirements Specification (SRS)

**Project:** LabelYourWindow
**Version:** 1.1.0
**Date:** 2026-04-16
**Platform:** macOS 15.0+ (Sequoia), Apple Silicon (arm64)

---

## 1. Overview

LabelYourWindow is a macOS menu bar application that displays translucent label overlays on windows to help users identify what each window is for in a multi-window workflow. Labels are auto-detected from window titles, customizable via rules or manual editing, and rendered as floating transparent overlays.

## 5. Architecture

```
WindowObserver (AXObserver + NSWorkspace)
    ↓ currentWindow changes (@Observable)
LabelManager (manual > rule > auto priority)
    ↓ label resolved
OverlayManager (NSPanel + animation)
    ↓ overlay shown/hidden
OverlayWindow (NSPanel) + LabelOverlayView (SwiftUI via NSHostingView)
```

**Key Files:**
- `LabelYourWindow/Core/WindowObserver.swift` - Window focus detection
- `LabelYourWindow/Core/LabelManager.swift` - Label resolution engine
- `LabelYourWindow/Core/OverlayManager.swift` - Overlay lifecycle and animation
- `LabelYourWindow/Core/SettingsManager.swift` - Settings persistence
- `LabelYourWindow/Views/Overlay/OverlayWindow.swift` - Floating panel
- `LabelYourWindow/Views/Overlay/LabelOverlayView.swift` - Label UI + glass effect
- `LabelYourWindow/Utilities/WindowTitleParser.swift` - Title parsing engine
